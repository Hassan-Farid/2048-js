INSTRUCTIONS:
============

1. The game consists of a 16 grid box containing a single number i.e. either 2 or 4 in the beginning
2. You have to move the different numbers using either up, down, left or right by either arrow keys or by moving the
   cursor up, down, left or right
3. If the numbers are same, then they are summed together into a single box, with the summed result added to score
4. If no other possible move can be taken, the game displays an alert message: "GAME OVER"
